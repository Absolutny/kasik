# 🎰 Demo Casino

Простое демонстрационное казино на Flask с 4 мини-играми.

## 🎮 Игры

1. **🎯 Игровые автоматы** - классические слоты с 3 барабанами
2. **♠️ Блэкджек 21** - карточная игра против дилера
3. **🪙 Подбрось монетку** - угадай орел или решка
4. **🎲 Кости** - брось кости против дилера

## 🚀 Установка

### Требования
- Python 3.8+
- pip

### Шаги установки

1. **Клонируйте репозиторий**
bash
git clone https://github.com/ваш-username/demo-casino.git
cd demo-casino

1. Создайте виртуальное окружение

bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate

1. Установите зависимости

bash
pip install -r requirements.txt

1. Запустите приложение

bash
python app.py

1. Откройте в браузере

http://localhost:5005

🔐 Регистрация и вход

1. Создайте аккаунт на странице регистрации
2. Войдите в систему
3. Начальный баланс: 1000 копеек
4. Для сброса баланса нажмите "🔄 Сбросить баланс"

🎯 Правила игр

Игровые автоматы

· 3 одинаковых символа = ×5 (×10 для 💎)
· 2 одинаковых символа = ×2

Блэкджек

· Цель: набрать 21 очко
· Блэкджек = ×2.5 выигрыш
· Дилер берет карты до 17

Монетка

· Угадай сторону монетки
· Выигрыш = ×2 ставки
· Шанс 50/50

Кости

· Брось 2 кубика против дилера
· Больше очков = победа
· Дубль (одинаковые числа) = ×3 выигрыш
